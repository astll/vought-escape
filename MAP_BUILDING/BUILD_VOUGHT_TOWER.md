# 🗺️ BUILD VOUGHT TOWER - Step by Step Map Guide

## Overview
Vought Tower is an 8-story building with multiple rooms and objectives. We'll build it using Roblox Studio's basic parts.

**Time to build:** ~30 minutes  
**Difficulty:** Beginner-friendly

---

## 🏢 Tower Dimensions

| Level | Name | Height |
|-------|------|--------|
| 1 | Lobby | 0-20 studs |
| 2-3 | Office Floors | 20-60 studs |
| 4-5 | Labs | 60-100 studs |
| 6 | Seven Meeting Room | 100-120 studs |
| 7 | Executive Offices | 120-140 studs |
| 8 | Rooftop | 140-160 studs |

---

## STEP 1: Delete the Baseplate

1. In **Explorer** (left panel), find **"Baseplate"**
2. Right-click it → **"Delete"**
3. You've cleared the space for Vought Tower!

---

## STEP 2: Create the Tower Base

1. In top toolbar, find **"Part"** button
2. Click **"Insert New Part"**
3. A gray cube should appear

**Rename and resize:**
1. Right-click part → **"Rename"** to **"TowerBase"**
2. In **Properties** panel:
   - **Size:** X: 50, Y: 160, Z: 50
   - **Position:** X: 0, Y: 80, Z: 0
   - **Color:** Dark gray

---

## STEP 3-10: Create Floors & Walls
