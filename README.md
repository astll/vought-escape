## 🎮 Game Overview

**7 Players, 1 Homelander**
- **6 Survivors (The Boys):** Collect virus vials, fill water tanks, escape Vought Tower
- **1 Homelander (Killer):** Hunt down survivors before they activate the sprinkler system

**Duration:** 10 minutes per match  
**Win Conditions:**
- 🟢 **Survivors Win:** All 8 water tanks filled → sprinklers activate → Homelander dies
- 🔴 **Homelander Wins:** All survivors eliminated
